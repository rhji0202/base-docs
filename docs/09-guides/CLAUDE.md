# Guides 폴더 가이드

이 폴더는 **개발자가 일상적으로 따르는 컨벤션·가이드**를 다룹니다.

## 폴더 구조
```
09-guides/
├── contributing.md       # 기여 방법, PR 절차
├── coding-standards.md   # 코딩 컨벤션
├── git-workflow.md       # 브랜치·커밋 전략
├── testing.md            # 테스트 전략
├── code-review.md        # 리뷰 가이드
├── claude-code-tips.md   # Claude Code 활용 팁
└── onboarding.md         # 신규 멤버 온보딩
```

## 가이드 vs 결정(ADR)
- **가이드**: "어떻게 일하는가" — 변경 빈번
- **ADR**: "왜 이렇게 결정했는가" — 불변

가이드 변경은 PR 리뷰만으로 가능, ADR은 별도 절차 필요.
