# Personas

> 타겟 사용자 정의.
> 기능 결정 시 "이 페르소나의 어떤 문제를 해결하는가"로 검증.

## 페르소나 작성 가이드

각 페르소나는 다음을 포함:
- 인구통계 (역할, 연령대, 기술 숙련도)
- 목표 (What they want)
- 좌절 (Pain points)
- 시나리오 (대표적 사용 맥락)
- 인용문 (실제 인터뷰에서)
